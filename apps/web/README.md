# apps/web — Nonogram Solver Web App

React + TypeScript + Vite によるWebフロントエンドです。WASMとして提供された求解エンジンをブラウザ上で動作させます。

## 前提条件

**このアプリを起動する前に、WASMパッケージのビルドが必要です。**
リポジトリルートの [README](../../README.md) を参照してください。

## コマンド

```bash
bun run dev      # 開発サーバー起動
bun run build    # プロダクションビルド
bun run test     # テスト実行
bun run lint     # Lint実行
```
